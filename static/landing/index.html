Community Solid Server

欢迎使用 Solid

本服务器实现了Solid 协议， 可以帮助你创建自己的Solid Pod 与身份。

面向用户的快速入门

注册账号 即可获取属于你的 Pod 和 WebID。

本服务器已禁用注册功能。 若计划对外开放，请务必更新根容器权限以防其他人修改你的数据。

想了解如何使用本服务器， 建议阅读 快速入门教程 以及官方文档。

面向开发者的快速入门

你可以使用 config 目录中的任意配置 来部署具备不同特性的服务器实例。 除了预置配置，还可以通过 配置生成器 自定义所需组件。

服务器支持将磁盘上的任意文件夹 暴露为根 Pod。
使用 --help 参数可查看更多命令行选项。

祝你拥有出色的 Solid 体验

欢迎访问 solidproject.org 了解 Solid。

我们诚挚邀请你 分享使用体验 ，或在遇到问题时 提交 Issue。

如何替换此页面

目前的首页是通过静态页面配置呈现的。 如需自定义，只需在 server 配置中加入如下条目： 

{
  "@type": "Override",
  "overrideInstance": { "@id": "urn:solid-server:default:RootStaticAsset" },
  "overrideParameters": {
    "@type": "StaticAssetEntry",
    "filePath": "/path/to/my/index.html"
  }
}
