## 用途分配

| 用途 | 境内 | 海外 |
|------|------|------|
| 主站 / API | `undefineds.cn` | `undefineds.co` |
| 用户 WebID | `alice.undefineds.cn` | `alice.undefineds.co` |
| Pod 存储 URL | `alice.xpod.cc` | `alice.xpod.cc` |
| DNS 托管 | DNSPod（腾讯云） | Cloudflare |

SP 域名必须与主域名隔离：用户 Pod 可存放任意内容，共享域名有被举报拉黑的风险。
